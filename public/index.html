NC News API

This API provides access to articles, topics, users and comments.

  • GET /api/topics
  • GET /api/articles
  • GET /api/articles/:article_id
  • GET /api/articles/:article_id/comments
  • POST /api/articles/:article_id/comments
  • PATCH /api/articles/:article_id
  • DELETE /api/comments/:comment_id
  • GET /api/users

GET /api/topics

Returns all topics.

Example Response:

{
  "topics": [
    { "slug": "coding", "description": "Code is love, code is life", "img_url": "https://..." }
  ]
}

GET /api/articles

Returns an array of articles.

Available Query Parameters:

  • sort_by - column to sort articles by
  • order - asc or desc
  • topic - filter articles by topic

Example Response (shortened):

{
  "articles": [
    {
      "article_id": 1,
      "title": "Running a Node App",
      "topic": "coding",
      "author": "butter_bridge",
      "comment_count": 17
    }
  ]
}

GET /api/articles/:article_id

Returns a single article by ID.

Example Request:

/api/articles/1

Example Response:

{
  "article": {
    "article_id": 1,
    "title": "Running a Node App",
    "author": "butter_bridge",
    "votes": 100
  }
}

GET /api/articles/:article_id/comments

Returns comments for the specified article.

Example Response:

{
  "comments": [
    {
      "comment_id": 18,
      "body": "Great read!",
      "author": "tickle122"
    }
  ]
}

POST /api/articles/:article_id/comments

Posts a new comment.

Request Body:

{
  "username": "butter_bridge",
  "body": "My comment text"
}

PATCH /api/articles/:article_id

Updates an article's vote count.

Request Body:

{
  "inc_votes": 1
}

DELETE /api/comments/:comment_id

Deletes a comment. Responds with no body.

GET /api/users

Returns all users.

Example Response:

{
  "users": [
    { "username": "butter_bridge", "name": "Jonny" }
  ]
}
